#OpenManus
OpenManus 一个简洁、开源的智能体实现方案。

## 项目概述
OpenManus 是开源 AI Agent 框架，专注：
- ReAct 模式：推理 + 行动循环
- 工具集成：浏览器/文件/代码执行/网络搜索
- MCP 协议：Model Context Protocol 远程工具服务
- 多 Agent 协作：规划流程 + 任务分解
- RL 调优：强化学习优化 Agent 性能

## 核心原则：
- 稳定性 > 性能 > 扩展性
- 异步优先 > 同步阻塞
- 最小改动 > 过度设计
