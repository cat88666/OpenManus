# OpenManus 五阶段开发计划

**项目**: 远程项目承包平台自动化系统 | **状态**: 进行中 (40%)

## 目标

多平台 → 爬取机会 → AI智能分析评分 → 人工审核 → 自动投递 → 获得订单

## 进度

| 阶段 | 目标 | 完成度 | 状态 |
|------|------|--------|------|
| 第一阶段 | 扩展大脑与记忆 | 100% | ✅ |
| 第二阶段 | 自定义Agent流程 | 100% | ✅ |
| 第三阶段 | 集成DeepSeek模型 | 0% | ⏳ |
| 第四阶段 | 强化浏览器自动化 | 0% | ⏳ |
| 第五阶段 | 人机交互界面 | 0% | ⏳ |

## 任务

- **第一阶段**: MySQL 读写能力 → DatabaseTool 类 | [详情](skills/tasks/01-第一阶段任务记录.md)
- **第二阶段**: ProjectHunterAgent → 三步工作流 (搜集→存储→评估) | [详情](skills/tasks/02-第二阶段任务记录.md)
- **第三阶段**: 多模型切换 → 复杂任务用R1，简单任务用快速模型 | [详情](skills/tasks/03-第三阶段任务记录.md)
- **第四阶段**: 真实爬取和投递 → 多平台支持、自动表单填充 | [详情](skills/tasks/04-第四阶段任务记录.md)
- **第五阶段**: 人工审核机制 → Streamlit 仪表板、状态管理 | [详情](skills/tasks/05-第五阶段任务记录.md)

## 架构

**系统流向**: 浏览器工具 → DatabaseTool → LLM模型 → ProjectHunterAgent → Streamlit仪表板

**数据流向**: 多平台职位 → 爬取 → 处理 → 数据库 → AI评估 → 仪表板 → 人工审查 → 投递 → 完成
